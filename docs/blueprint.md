# **App Name**: ConnectMe

## Core Features:

- User Profiles: Allow users to create and customize their profiles with personal information, photos, and bios.
- Post Creation: Enable users to create and share text, images, and video posts with their network.
- News Feed: Display a personalized news feed of posts from friends, followed accounts, and relevant communities.
- Friending and Following: Allow users to connect with friends and follow interesting accounts to expand their network.
- Comments and Reactions: Enable users to comment on posts and react with likes or other emojis.
- AI-Powered Content Suggestions: Suggest content to users based on their profile data using an AI tool to analyze trends in their posts.

## Style Guidelines:

- Primary color: Saturated blue (#3F51B5) to inspire trust and connection.
- Background color: Light blue (#E8EAF6), creating a calm and inviting atmosphere.
- Accent color: Orange (#FF9800) to highlight key actions and calls to action.
- Headline font: 'Poppins', sans-serif. Body font: 'PT Sans', sans-serif, creating a modern and readable interface. Note: currently only Google Fonts are supported.
- Use a set of consistent and recognizable icons for navigation and common actions.
- Employ a clean and intuitive layout with clear hierarchy and consistent spacing.
- Incorporate subtle animations for transitions and user interactions to enhance engagement.
